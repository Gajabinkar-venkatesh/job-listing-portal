# Job Listing Portal

A Full Stack Job Listing Portal built using React.js, Node.js, Express.js, and MongoDB.

## Features
- User Registration and Login
- JWT Authentication
- Job Posting
- Apply for Jobs
- Admin Dashboard
- Secure API routes

## Technologies Used
Frontend:
- React.js
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT

## Project Structure
- frontend/
- backend/

## How to Run

Backend:
cd backend
npm install
npm start

Frontend:
cd frontend
npm install
npm start
